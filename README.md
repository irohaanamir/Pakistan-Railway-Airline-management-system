# 🚂 Pakistan Railway Management System

A comprehensive console-based management application developed during my 1st semester to automate core railway and airline  operations. This project focuses on implementing pure procedural programming logic, structuring clean dataflows, and practicing efficient memory management using C++.As this was my first-semester coding project developed alongside a classmate, a few minor mistakes occurred. However, we are actively working to improve our coding logic and eliminate errors through consistent practice.
## 🚀 Key Features
* **Ticket Booking & Cancellation:** Allows passengers to reserve seats and cancel bookings.
* **Train Scheduling:** Real-time checking and updating of train timings, routes, and available seats.
* **Passenger Database:** Handles passenger records and updates travel logs.
* **Admin Dashboard:** Secure portal for staff to update train routes and manage sales.

## 🛠️ Concepts & Technologies Used
* **Language:** C++
* **Core Logic:** 1D character arrays separated by local \0 null-terminators,Nested loop,Entity state transitions, Structures/Classes,Arrays, and custom logic setups (without heavy external libraries).

## 💻 How to Run the Project
1. Clone or download this repository to your computer.
2. Open the project folder in any C++ IDE (like Dev-C++, VS Code, or Code::Blocks).
3. Compile and run the `main.cpp` file.
